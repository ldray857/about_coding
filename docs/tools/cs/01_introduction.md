# 🖥 我的电脑管理工具推荐

欢迎来到我的电脑管理工具页面，这里整理了我常用的各类工具，帮助你更高效地管理系统、开发环境和工作流程。每个工具都有简短描述，方便快速了解。

---

## ⚡ 系统管理

- 🧹 **CCleaner** – 系统清理与优化，删除垃圾文件、管理启动项。  
- 🛡 **Windows Security / Defender** – 系统防护，病毒和恶意软件检测。  
- 🗂 **Everything** – 超快速文件搜索工具，让你瞬间找到电脑里的文件。  
- 🖥 **Process Explorer** – 进程监控工具，替代任务管理器，详细查看 CPU、内存占用。  

## 💾 磁盘与备份

- 📦 **Macrium Reflect** – 系统和数据备份工具，支持镜像和恢复。  
- 🧰 **TreeSize Free** – 磁盘空间分析，帮你快速找出大文件。  
- 🔄 **SyncToy / FreeFileSync** – 文件同步工具，方便多台电脑间保持文件一致。  

## 🌐 网络与下载

- 🌍 **Wireshark** – 网络抓包与分析工具，查看网络流量和故障排查。  
- ⬇ **Internet Download Manager (IDM)** – 高速下载工具，支持断点续传。  
- 🖧 **PuTTY** – SSH/Telnet 客户端，方便远程管理服务器。  

## 🖌 系统美化与个性化

- 🎨 **Rainmeter** – 桌面美化和信息面板工具，可显示 CPU、内存、网络状态。  
- 🔔 **F.lux** – 调整屏幕色温，保护眼睛，尤其适合夜间工作。  

## 🛠 开发与工具管理

- 🐍 **Anaconda / Miniconda** – Python 环境和包管理工具。  
- 💻 **VS Code** – 多语言代码编辑器，可通过插件管理开发环境。  
- 📦 **Chocolatey / Winget** – Windows 软件包管理工具，一条命令即可安装、更新软件。  

## 🧰 文件与资料管理

- 📁 **Total Commander** – 双窗口文件管理器，操作高效。  
- 🗂 **Notion / Obsidian** – 资料整理和笔记管理工具，方便知识库搭建。  
- 🖼 **ShareX** – 屏幕截图、录屏和分享工具，一体化管理图片和视频。
