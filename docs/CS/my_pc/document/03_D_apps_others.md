## 3.应用详解
### 3.8 2021.11-supermap
一个**由超图软件开发且在国产 GIS 领域领先**的大型地理信息系统软件。

**主要用处：**
* **国产 GIS 开发：** 进行地理空间数据的处理、分析及地图制图，是国内政务、国土等项目的常用工具。
* **三维可视化：** 支持超大规模三维数据的渲染与分析。

**优点：**
* **本地化支持强：** 对国内坐标系和标准支持极好。
* **大数据处理：** 针对国产软硬件环境进行了深度优化。

**注意事项：**
* **资源占用：** 运行大型工程时非常吃内存。
* **学习曲线：** 相比 QGIS，其操作逻辑更偏向专业开发，新手需要时间适应。

**个人感受：**
国产 GIS 的“排面”，虽然在某些交互上还有提升空间，但在国内工程落地方面是无可替代的。

### 3.9 CMake
一个**开源且跨平台的**自动化建构系统。

**主要用处：**
* **项目编译配置：** 管理 C/C++ 项目的编译过程，生成本地的 Makefile 或 VS 工程文件。
* **多平台开发：** 确保同一份代码在 Windows、Linux 和 macOS 下都能顺利编译。

**优点：**
* **行业标准：** 绝大多数 C/C++ 开源库（如 OpenCV）都使用它。
* **灵活强大：** 支持复杂的逻辑判断和外部库查找。

**注意事项：**
* **语法生涩：** `CMakeLists.txt` 的语法比较独特，初学者容易写错。

**个人感受：**
C/C++ 开发者的“刚需”，虽然配置的时候很痛苦，但一旦配好，后续的跨平台编译就非常省心了。



---

### 3.10 git
一个**开源且分布式**的代码版本控制系统。

**主要用处：**
* **代码版本管理：** 记录代码的每一次修改，支持随时“回滚”到之前的版本。
* **团队协作：** 配合 GitHub/Gitee 实现多人共同开发一个项目。

**优点：**
* **分支管理：** 极其强大的分支（Branch）功能，让你可以在不影响主线代码的情况下尝试新想法。
* **安全可靠：** 所有的修改记录都保存在本地和远程，极难丢失数据。

**注意事项：**
* **命令行门槛：** 常用指令（push, pull, commit, merge）需要熟练掌握，初学者容易遇到“冲突”。

**个人感受：**
程序员的“后悔药”。它是每个开发者的必修课，哪怕只是自己写作业，养成 Git 提交的习惯也能避免代码意外丢失。



---

### 3.11 qgisapp (QGIS)
一个**开源且功能全面**的跨平台桌面地理信息系统。

**主要用处：**
* **地图编辑与分析：** 浏览、编辑各种格式的矢量和栅格数据。
* **制图输出：** 制作专业级的高质量地图。

**优点：**
* **完全免费：** 作为开源软件，它拥有极丰富的插件生态（Plugin）。
* **轻量高效：** 相比 ArcGIS Pro，它启动更快，对电脑配置要求相对稍低。

**注意事项：**
* **插件兼容性：** 有些第三方插件更新不及时，可能会导致软件报错。

**个人感受：**
GIS 界的“自由之光”。如果你不想折腾昂贵的授权，QGIS 是最强大的免费替代方案，尤其适合处理 OpenStreetMap 等开源数据。



---

### 3.12 spss_app
一个**专业且广泛应用于社科研究**的统计分析软件。

**主要用处：**
* **数据统计分析：** 问卷调查结果处理、回归分析、方差分析等。
* **科研绘图：** 生成符合论文标准的统计图表。

**优点：**
* **操作简便：** 典型的“菜单式”操作，不需要写代码就能完成复杂的统计。
* **权威性：** 其输出结果被全球学术期刊广泛认可。

**注意事项：**
* **灵活性较低：** 相比 Python 或 R 语言，它的自定义扩展能力较弱。

**个人感受：**
处理非空间统计数据的“神器”。如果你在做 GIS 相关的社会科学研究，SPSS 会比自己写 Python 脚本快得多。



---

### 3.13 vscode (Visual Studio Code)
一个**由微软开发且插件生态极丰富**的跨平台源代码编辑器。

**主要用处：**
* **全栈开发：** 写 HTML/CSS/JS、Python、C++ 或 Markdown 笔记。
* **远程开发：** 配合 SSH 插件连接远程服务器进行代码编辑。

**优点：**
* **极速启动：** 虽然功能强大，但启动速度远快于专业 IDE。
* **无限扩展：** 只要装对插件，它几乎能变成任何语言的编译器。

**注意事项：**
* **配置成本：** 对于 C/C++ 等语言，需要自己配置环境（如编译器路径），不如 Dev-C++ 那么“傻瓜”。

**个人感受：**
现代开发的“瑞士军刀”。正如你之前尝试配置环境一样，虽然初期有点折腾，但一旦调顺了，它是目前最好用的编辑器，没有之一。



---

### 3.14 zjuasi (ZJU-Asia/Campus Network)
一个**专为浙大师生设计**的校园网接入或认证辅助工具。

**主要用处：**
* **网络认证：** 协助完成校内网络的登录与身份验证。
* **访问校内资源：** 确保在寝室或实验室能正常访问浙大内网资源。

**个人感受：**
紫金港生活的“通行证”。只要在校内用网，它就是必不可少的底层工具。
