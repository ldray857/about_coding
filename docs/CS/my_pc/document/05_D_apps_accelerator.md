# D盘文件夹 /apps_accelerator 说明

## 1.文件目录

- **aria2-1.37.0-win-64bit-build1**  
- **BaiduPCS-Go-v4.0.0-windows-x86** 
- **FlClash** 

## 2.相应应用

- **aria2**   
极其轻量级、多协议的命令行下载工具  
- **BaiduPCS-Go**  
仿 Linux 终端格式的百度网盘命令行客户端。
- **FlClash**  
基于 Clash 内核的、使用 Flutter 开发的图形化网络代理工具。 

## 3.应用详解

### 3.1aria2

它是目前地球上最强大的命令行下载引擎之一，支持 HTTP、FTP、BT 种子和磁力链接。

**主要用处：**

- 作为下载底层引擎（很多浏览器插件如“油猴”脚本会调用它来满速下载）  
- 下载超大型文件（如几百 GB 的 DEM 遥感数据、Linux 镜像）  

**优点：**

- **极度轻量：** 运行时几乎不占内存和 CPU，非常适合后台静默运行  
- **多线程加速：** 能把一个文件的下载任务拆分成几十个片段同时下载，突破服务器单线程限速  

**注意事项：**

- **无图形界面：** 它是纯命令行的，通常需要配合 WebUI（如 AriaNg）使用  
- **磁盘预分配：** 下载大文件时会预先占用磁盘空间，如果 D 盘空间本来就紧，容易报错  
- **残留文件：** 未完成的任务会生成 .aria2 后缀的临时文件，清理时记得一并删掉  

**个人感受：**  
- 搭配ai帮助，下载文件如虎添翼，突破线程的限制



### 3.2BaiduPCS-Go

**主要用处：**

- 在不安装臃肿的“百度网盘官方版”的情况下，实现文件的极速上传和下载。


**优点：**

- **简洁高效：** 告别官方版的广告和弹窗，内存占用极低。
- **速度较快：** 在特定配置下，它的下载效率往往高于官方限速状态。
- **无图形界面：** 它是纯命令行的。

**注意事项：**

- 账号封禁风险： 频繁使用第三方工具大流量下载，可能会触发百度网盘的风险控制，导致账号被临时限速。


**个人感受：**  
- 没有恶心人的百度网盘广告。并且，它可以让你直接找到那些下载下来的东西，下载了多少就可以打开多少




### 3.3FlClash

**主要用处：**  
- 科学上网，不言而喻

**个人感受：**  
- 有点小贵，但比其他的大多数都要稳定，毕竟是老牌机场
